# STAT 184 - Github Practice Repository for HW #4.3 - Reproducible Data Analysis

## Information About this Repository

This repository is for Homework 4.4 in my Stat 184 course. The primary goal is to practice using github to better understand how to use it. The repository contains the files from my Homework 4.3 assignment on reproducible data analysis. The files are written in quarto and produce a PDF report of several data visualizations we had made over the course of the semester. You will also find a plan document as well as this readme file.

## Data Sources

The analysis for the PDF report of data visualizations uses the following data:

   - Busiest airport data: Data was scraped from Wikipedia's List of Busiest Airports by Passenger Traffic.
   - The Monte Carlo simulation data comes from simulated samples that were generated inside of R
   - The Ai prompting section uses data from the calcium.csv dataset, which was provided to Chatgpt to create two plots
